## Slide 3: 场景背景

### 为什么需要智能投资 Agent？

**传统投资痛点：**
- ❌ 数据分散：多个平台切换
- ❌ 分析复杂：需要专业知识
- ❌ 决策主观：情绪影响判断
- ❌ 效率低下：手动计算耗时

**我们的解决方案：**
- ✅ 一站式数据获取（仅 akshare，国内可用）
- ✅ 自动化分析（6 种经典策略）
- ✅ 客观决策（量化优化）
- ✅ 高效执行（一键生成报告）

**目标用户：**
- 个人投资者
- 投资顾问
- 金融专业学生

```
                  │

    │              │              │

```

**核心模块：**
1. DataManager - 数据获取（仅 akshare）

### 多 API 整合与交叉验证

**数据源（仅国内可用）：**
- akshare - A 股历史数据
- 东方财富 - 财务指标、宏观数据

**交叉验证实现：**
```python
def fetch_stock_financials(self, stock_code: str) -> dict:
    # API 1: akshare 获取财务指标
    # API 2: 东方财富获取估值指标
    return {
        'akshare': {'ROE': 15.2%, '毛利率': 92.1%},
        'eastmoney': {'PE': 35.2, 'PB': 12.5}
    }
```

**网络稳定性：**
- 3 次重试机制
- 指数退避（2^attempt 秒）
- 数据缓存
