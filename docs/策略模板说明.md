# 投资策略模板 - 运行说明

## 📋 策略列表

本模板包含 6 种经典投资策略：

| 策略 | 英文名 | 目标 | 适合人群 |
|------|--------|------|---------|
| **等权重配置** | Equal Weight | 每个资产相同权重 | 新手、懒人投资 |
| **马科维茨最优** | Markowitz Optimal | 最大化夏普比率 | 追求风险收益比 |
| **最小方差配置** | Minimum Variance | 最小化波动率 | 风险厌恶型 |
| **全天候配置** | All Weather / Risk Parity | 风险平价 | 长期稳健投资 |
| **最大分散化** | Maximum Diversification | 最大化分散比率 | 追求分散风险 |
| **市场组合** | Market Portfolio | 市值加权 | 被动投资 |

## 📊 策略详解

### 策略 1: 等权重配置 (Equal Weight)

**最简单策略** - 每个资产分配相同权重

```python
权重 = [20%, 20%, 20%, 20%, 20%]  # 5 只股票
```

**优点：**
- 简单易懂
- 自动再平衡
- 无需预测

**缺点：**
- 不考虑风险差异
- 不考虑相关性

---

### 策略 2: 马科维茨最优 (Markowitz Optimal)

**经典策略** - 最大化夏普比率

```python
目标：max (R_p - R_f) / σ_p

约束:
  Σw_i = 1
  0 ≤ w_i ≤ 1
```

**优点：**
- 理论完善
- 风险收益最优

**缺点：**
- 对输入敏感
- 可能集中度高

---

### 策略 3: 最小方差配置 (Minimum Variance)

**保守策略** - 最小化组合波动率

```python
目标：min σ_p² = w'Σw

约束:
  Σw_i = 1
  0 ≤ w_i ≤ 1
```

**优点：**
- 风险最低
- 适合熊市

**缺点：**
- 收益可能较低
- 可能过度保守

---

### 策略 4: 全天候配置 (All Weather / Risk Parity)

**桥水基金策略** - 风险平价

```python
目标：每个资产风险贡献相等

风险贡献_i = w_i × 边际风险_i
目标：风险贡献_1 = 风险贡献_2 = ... = 风险贡献_n
```

**优点：**
- 真正的分散
- 适应各种市场环境

**缺点：**
- 计算复杂
- 需要杠杆（理论上）

---

### 策略 5: 最大分散化配置 (Maximum Diversification)

**优化策略** - 最大化分散化比率

```python
分散化比率 = (Σw_i × σ_i) / σ_p

目标：max 分散化比率
```

**优点：**
- 充分利用分散化
- 不依赖预期收益

**缺点：**
- 可能偏向低波动资产

---

### 策略 6: 市场组合 (Market Portfolio)

**被动策略** - 按市值加权

```python
权重_i = 市值_i / 总市值
```

**优点：**
- 代表市场
- 被动投资

**缺点：**
- 可能过度集中
- 需要市值数据

## ⚠️ 注意事项

### 1. 数据源

**仅使用 akshare** - 国内可用 API

```python
import akshare as ak

df = ak.stock_zh_a_hist(
    symbol="600519",
    period="daily",
    adjust="qfq"  # 前复权
)
```

### 2. 网络问题

akshare 有时网络不稳定，代码已添加重试机制：

```python
# 自动重试 3 次
for attempt in range(3):
    try:
        df = ak.stock_zh_a_hist(...)
        break
    except:
        time.sleep(2 ** attempt)  # 指数退避
```

### 3. 数据质量

- 使用前复权数据（`adjust="qfq"`）
- 建议至少 120 个交易日（约半年）
- 剔除停牌股票

### 4. 模型假设

- 收益率服从正态分布
- 不允许做空（权重 0-1）
- 无交易成本
- 历史数据不代表未来
