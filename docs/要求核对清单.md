## ⚠️ 注意事项

### 1. API 限制说明
akshare 有时网络不稳定，已添加：
- 3 次重试机制
- 指数退避（2^attempt 秒）
- 数据缓存

### 2. 数据质量
- 使用前复权数据（adjust="qfq"）
- 建议至少 120 个交易日
- 剔除停牌股票

### 3. 模型假设
- 收益率正态分布
- 不允许做空
- 无交易成本
- 历史不代表未来
