## 🚀 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行主程序
python src/investment_agent.py

# 3. 运行策略模板
python src/portfolio_strategies.py
```

---

## 📊 6 种投资策略

1. **等权重配置** - 简单分散
2. **马科维茨最优** - 最大化夏普比率
3. **最小方差配置** - 最小化风险
4. **全天候配置** - 风险平价
5. **最大分散化** - 最大化分散比率
6. **市场组合** - 市值加权
